## AI as Your Web Developer

Imagine simply telling your computer: "Create a team page with our employees." And it does. That's exactly what Claude Code enables.

### What is Claude Code?

Claude Code is an AI-powered development tool that understands your code. It can create pages, update content, and adjust designs — all at your request.

### The Workflow

1. Describe what you want
2. Claude Code implements it
3. You review the result
4. Done — your website is updated
